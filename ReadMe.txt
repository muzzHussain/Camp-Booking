# Camp-Booking

## This Zip file contains 3 folder
1) BookingWeb -> It contains the frontend of the application.

2) CampBookingAPI -> It contains the backend of the application.

3) DB Script -> It contains the DB Script which is used in the application.

## To Run this application:
1) Restore the Database which is in the DBScript folder.
2) After restore, run the API in the CampBookingAPI Folder
3) After that run the angular application using ng serve.
